Python 数字类型

Python 数字中有四种类型：整数，布尔型，浮点型，复数类型
int 整数，只有一种类型，就是长整型
bool：True
浮点数：如1.23
complex 复数类型 1+2j

Python 字符串类型

使用三个引号可以指定一个多行字符串
反斜杠可以用来转义，可以使用r来使用使反斜杠不转义
字符串可以用+来连接字符串，使用*运算符重复

python print 默认使换行的 可以在后面加上 end='' 来去掉换行

print("python test",end='')

python argv,path，platform

sys.argv 命令行的参数，包括脚本名称
sys.path 查找模块所在的目录列表
sys.platform 查找运行平台的系统 win or linux ...

number 函数

exp()函数

exp函数 返回x的指数 exp()是不能之际访问的，需要导入math模块通过静态调用该方法

floor函数

floor(x)返回数字的下舍整数 也就是小于等于x 通过静态调用该方法
floor()是不能直接访问的，需要导入Math模块，通过静态对象调用该方法

log 函数

log()函数

log(x)方法返回以e为基底的x的对数 log(x1,x2)返回以x2为基底的x1的对数 log10(x)返回以10为基底的x的对数

max min 函数

max()函数 返回指定参数的最小值，参数可以为序列，入参类型要一致
min 函数返回指定参数的最小值，参数可以为序列

入参是序列的话，单序列入参返回序列中最大的一个数值，多序列入参，按索引顺序逐一对比各序列的当前索引的值，直到遇见最大值立即停止对比，返回最大值所在序列

modf pow 函数

math.modf()函数 返回x的整数部分与小数部分，两部分的数值符号与x相同，整数部分以浮点型表示。
pow() 返回x**y运算后的值
内置pow()调用会把参数作为整型，而math模块则会把参数转换成float.
